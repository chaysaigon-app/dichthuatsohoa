Hướng dẫn Firebase

Thiết lập Firebase để website dichthuatsohoa.com có tính năng đăng nhập thành viên thật sự. Toàn bộ miễn phí — không cần biết lập trình, chỉ cần làm theo từng bước.

Bước 1 — Tạo tài khoản & project Firebase

1

Vào trang Firebase

Mở trình duyệt → gõ vào thanh địa chỉ: firebase.google.com

2

Đăng nhập bằng Gmail

Bấm nút Get started → đăng nhập bằng tài khoản Google / Gmail của công ty (khuyến nghị dùng sohoavn@gmail.com)

3

Tạo project mới

Bấm Add project → đặt tên: dichthuatsohoa → bấm Continue

4

Tắt Google Analytics

Bấm công tắc để tắt Google Analytics (không cần thiết) → bấm Create project

5

Chờ tạo xong

Chờ khoảng 20-30 giây → khi thấy dấu ✅ → bấm Continue

Bước 2 — Bật tính năng đăng nhập (Authentication)

1

Vào Authentication

Ở menu bên trái → bấm Build → bấm Authentication

2

Get started

Bấm nút Get started

3

Bật Email / Mật khẩu

Bấm vào Email/Password → bấm công tắc Enable (cái đầu tiên) → bấm Save

4

Bật đăng nhập Google (tuỳ chọn)

Bấm Google → bật Enable → chọn email hỗ trợ sohoavn@gmail.com → bấm Save

Kết quả: Từ giờ, khách hàng có thể đăng ký và đăng nhập vào website bằng email và mật khẩu — hoặc bằng tài khoản Google.

Bước 3 — Tạo Firestore Database (lưu thông tin thành viên)

1

Vào Firestore

Menu bên trái → Build → Firestore Database

2

Create database

Bấm Create database

3

Chọn Test mode

Chọn Start in test mode → bấm Next

4

Chọn vùng Singapore

Chọn asia-southeast1 (Singapore — gần nhất với Việt Nam, tốc độ tốt nhất) → bấm Enable

Bước 4 — Lấy mã cấu hình Firebase

1

Vào Project Settings

Bấm biểu tượng ⚙️ bánh răng bên cạnh chữ "Project Overview" ở menu trái → chọn Project settings

2

Thêm ứng dụng Web

Cuộn xuống mục Your apps → bấm biểu tượng </> (Web app)

3

Đặt tên app

Gõ tên: DichThuatSoHoa Web → bấm Register app

4

Copy đoạn mã

Sẽ hiện ra đoạn mã có apiKey, authDomain, projectId... → bấm Copy → lưu vào Notepad → bấm Continue to console

⚠ Quan trọng: Đoạn mã cấu hình này là "chìa khóa" kết nối website với Firebase. Giữ lại cẩn thận trong Notepad. Đừng chia sẻ cho người lạ.

Bước 5 — Dán mã vào file index.html và admin.html

Mở từng file bằng Notepad (nhấp chuột phải → Open with → Notepad), nhấn Ctrl+F, tìm dòng </body>, dán đoạn code bên dưới vào NGAY TRƯỚC dòng đó:

ĐOẠN MÃ CẦN DÁN VÀO — thay phần màu cam bằng mã của bạn 
<!-- Firebase SDK – dán ngay trước </body> -->
<script type="module">
  import { initializeApp } from
    "https://www.gstatic.com/firebasejs/10.7.0/firebase-app.js";
  import { getAuth, signInWithEmailAndPassword,
           createUserWithEmailAndPassword, signOut, onAuthStateChanged }
    from "https://www.gstatic.com/firebasejs/10.7.0/firebase-auth.js";
  import { getFirestore, doc, setDoc, getDoc }
    from "https://www.gstatic.com/firebasejs/10.7.0/firebase-firestore.js";

  // ← DÁN MÃ CẤU HÌNH CỦA BẠN VÀO ĐÂY (từ bước 4)
  const firebaseConfig = {
    apiKey:            "AIza...",              // thay bằng mã của bạn
    authDomain:        "dichthuatsohoa-xxx.firebaseapp.com",
    projectId:         "dichthuatsohoa-xxx",
    storageBucket:     "dichthuatsohoa-xxx.appspot.com",
    messagingSenderId: "123456789",
    appId:             "1:123456789:web:abcdef"
  };

  const app  = initializeApp(firebaseConfig);
  const auth = getAuth(app);
  const db   = getFirestore(app);

  // Đăng ký thành viên mới
  window.registerUser = async (email, password, name, phone) => {
    const cred = await createUserWithEmailAndPassword(auth, email, password);
    await setDoc(doc(db, "users", cred.user.uid), {
      name, email, phone,
      role: "member",
      createdAt: new Date().toISOString()
    });
    return cred.user;
  };

  // Đăng nhập
  window.loginUser = async (email, password) => {
    return await signInWithEmailAndPassword(auth, email, password);
  };

  // Đăng xuất
  window.logoutUser = () => signOut(auth);

  // Theo dõi trạng thái đăng nhập
  onAuthStateChanged(auth, user => {
    const loginBtn  = document.querySelector('.nbtn');
    if (user && loginBtn) {
      loginBtn.textContent = '👤 ' + (user.displayName || user.email);
    }
  });
</script>
Lưu ý: Làm bước này cho cả 2 file: index.html và admin.html. Sau đó lưu file (Ctrl+S) và push lên GitHub. Vercel sẽ tự cập nhật.

Bước 6 — Kiểm tra và bảo mật Firestore

1

Test đăng ký

Vào website → bấm Đăng ký → điền email và mật khẩu → bấm Đăng ký ngay. Nếu thành công sẽ hiện thông báo.

2

Kiểm tra trong Firebase Console

Vào Firebase Console → Authentication → tab Users → sẽ thấy tài khoản vừa đăng ký xuất hiện ở đây.

3

Bảo mật Firestore (sau 30 ngày)

Vào Firestore Database → tab Rules → thay toàn bộ nội dung bằng đoạn sau để giới hạn quyền truy cập:

FIRESTORE SECURITY RULES — dán vào tab Rules
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId} {
      // Chỉ người dùng đó mới được đọc/ghi dữ liệu của mình
      allow read, write: if request.auth != null
                         && request.auth.uid == userId;
    }
  }
}
